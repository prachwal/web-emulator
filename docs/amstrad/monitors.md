# Amstrad CPC — Monitory

Zgodność: DEFv1

## GT-65 (zielony)

| Właściwość | Wartość |
|---|---|
| Typ | CRT monochromatyczny |
| Fosfor | Zielony P1 |
| Przekątna | 12" |
| Proporcje | 4:3 |
| Wejście | RGB DIN (6-pin, mono) |
| Kompatybilność | CPC 464, CPC 664, CPC 6128 |

Zielony monitor dostarczany z CPC 464 w tańszej konfiguracji. Wyświetla wszystkie kolory jako odcienie zieleni.

## CTM-644 (kolor)

| Właściwość | Wartość |
|---|---|
| Typ | CRT kolorowy |
| Fosfor | RGB (kolorowa maska) |
| Przekątna | 14" |
| Proporcje | ~4:3 |
| Wejście | RGB DIN (6-pin) |
| Kompatybilność | CPC 464, CPC 664, CPC 6128 |

Standardowy kolorowy monitor CPC. Obsługuje tryby 160×200 16c, 320×200 4c, 640×200 2c.

## MP-1 (RF modulator)

| Właściwość | Wartość |
|---|---|
| Typ | RF modulator TV (PAL) |
| Kompatybilność | CPC 464, CPC 664, CPC 6128 |

Pozwala podłączyć CPC do standardowego telewizora PAL.

## W emulatorze

Zobacz: [Amstrad CPC — opis maszyny](cpc.md)

Wybór monitora w menu: **Amstrad → model → Monitor → Text/Bitmap → rozdzielczość**.
Monitory monochromatyczne (GT-65) automatycznie konwertują paletę na odcienie zieleni.
