# Commodore 128

Zgodność: DEFv1

## Specyfikacja video

| Właściwość | Wartość |
|---|---|
| CPU | 8502 + Z80 @ 2 MHz |
| RAM | 128 KB |
| Chip VDC | 8563 (80-kolumnowy tryb tekstowy) |
| Chip VIC-II | 6569 (C64-kompatybilny, 40-kolumnowy) |
| Rozdzielczość VDC | 640 × 200 (lub 640 × 400 interlaced) |
| Rozdzielczość VIC-II | 320 × 200 |
| Font VDC | 8 × 16 (256 glyphów) |
| Font VIC-II | 8 × 8 (C64 CHARGEN) |
| OS | CP/M 3.0 + BASIC 7.0 |

## Co już mamy

- [x] Preset `c128-vdc-80x25` — tryb VDC 80×25 (640×200, cell 8×8, PAR 5/6)
- [x] Preset `c128-vdc-80x25-interlace` — tryb VDC interlaced 80×25 (640×400, cell 8×16, PAR 5/6)
- [x] Oryginalny font VDC z ROM-u C128 (Zimmers c128.bin offset 0x1800, 2048B, LSB-first)
- [x] Demo z BASIC listingiem
- [x] Preset `c128-vicii-40x25` — tryb 40-kolumnowy VIC-II (320×200)
- [x] Preset `c128-vicii-320x200` — tryb bitmapowy VIC-II Hi-Res
- [x] Preset `c128-vicii-160x200` — tryb bitmapowy VIC-II Multicolor
