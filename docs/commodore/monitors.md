# Commodore — Monitory

Zgodność: DEFv1

## Commodore 1701 / 1702

| Właściwość | Wartość |
|---|---|
| Typ | CRT kolorowy |
| Fosfor | RGB (maska), 0.64mm dot pitch |
| Przekątna | 13" |
| Proporcje | 4:3 |
| Wejście | Composite video (1701), Luma/Chroma 8-pin (1702) |
| Kompatybilność | C64, VIC-20, PET (z adapterem) |

1701 i 1702 są prawie identyczne — różnica w złączu wejściowym.
S-Video w 1702 daje ostrzejszy obraz.

## NTSC TV / PAL TV

Standardowy telewizor CRT (14") podłączany przez RF modulator (VIC-20, C64).
