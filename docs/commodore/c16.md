# Commodore 16 / 116

Zgodność: DEFv1

## Specyfikacja video

| Właściwość | Wartość |
|---|---|
| CPU | 7501 @ 1.77 MHz (NTSC) / 8870 @ 1.77 MHz |
| RAM | 16 KB (C16) / 32 KB (C116) |
| Chip TED | 7360 (video, audio, I/O) |
| Rozdzielczość | 320 × 200 (standard), 160 × 200 (multicolor) |
| Tekst | 40 × 25 |
| Font | 8 × 8 (256 glyphów, PETSCII order) |
| Kolory | 128 (8 odcieni × 8 kolorów × 2 zestawy) |
| Storage | Datasette / kartridż |

## Paleta TED (128 kolorów)

Paleta TED 7360: 8 barw × 8 poziomów luminancji + macierz alternatywna (bit 7).
Indeksy 0-63: macierz standardowa, 64-127: macierz alternatywna.

Pierwsze 16 kolorów (jak VIC-II):
0: #000000, 1: #FFFFFF, 2: #880000, 3: #AAFFEE,
4: #CC44CC, 5: #00CC55, 6: #0000AA, 7: #EEEE77,
8: #FF8855, 9: #FF00AA, 10: #885500, 11: #555555,
12: #888888, 13: #00FF00, 14: #0000FF, 15: #BBBBBB

Pełna paleta 128 kolorów zdefiniowana w `presets/c16.ts`.

## Ekran startowy (REAL)

```
   **** COMMODORE 16 ****
    16K RAM SYSTEM
     12277 BASIC BYTES FREE

READY.
```

Biały tekst na czarnym tle. Procesor TED 7360, 40×25 znaków.
