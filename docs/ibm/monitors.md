# IBM — Monitory

Zgodność: DEFv1

## IBM 5151 (MDA)

| Właściwość | Wartość |
|---|---|
| Typ | CRT monochromatyczny |
| Fosfor | Zielony (P1) |
| Przekątna | 12" |
| Proporcje | 4:3 |
| Wejście | TTL (9-pin DE-9) |
| Kompatybilność | IBM MDA, Hercules HGC |

Monochromatyczny monitor dla MDA. Rozdzielczość 720×350, 80×25 znaków.

## IBM 5153 (CGA)

| Właściwość | Wartość |
|---|---|
| Typ | CRT kolorowy |
| Fosfor | RGB, 0.43mm dot pitch |
| Przekątna | 12" |
| Proporcje | ~4:3 |
| Wejście | TTL RGBI (9-pin DE-9) |
| Kompatybilność | IBM CGA |

Kolorowy monitor CGA. Rozdzielczość 640×200 (2 kolory) lub 320×200 (4 kolory).
