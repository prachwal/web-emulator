# Sinclair — Monitory

Zgodność: DEFv1

## PAL TV (RF)

| Właściwość | Wartość |
|---|---|
| Typ | CRT kolorowy |
| Fosfor | RGB (maska) |
| Przekątna | 14"-21" |
| Proporcje | 4:3 |
| Wejście | RF (antenowe) / Composite video |
| System | PAL 50Hz |

Standardowy telewizor PAL używany z ZX Spectrum w Europie. Obraz 256×192 wyśrodkowany z borderem (24+192+24 linii). 

## Green Monitor (monochromatyczny)

Rzadko używany z Spectrum — głównie w profesjonalnych konfiguracjach.
