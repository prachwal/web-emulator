# ZX Spectrum 128 / +3

Zgodność: DEFv1

## Specyfikacja video

| Właściwość | Wartość |
|---|---|
| CPU | Z80 @ 3.5 MHz |
| RAM | 128 KB |
| Video | Ten sam ULA co 48K (256×192, 16 kolorów) |
| Dźwięk | AY-3-8912 (3 kanały) |
| Storage | Kaseta (128) / 3" floppy (+3) |
| OS | Sinclair BASIC 128 + CP/M (+3) |

## Co już mamy

- [x] Preset `zx128-text-32x24` — 256×192, 16 kolorów
- [x] Font z ZX Spectrum ROM
- [x] Demo z BASIC listingiem, informacjami o 128K
- [x] Preset `zx128-attr-256x192` — tryb atrybutowy 256×192 (taki sam jak 48K)
- [x] Paleta 128K — jaśniejsze kolory (inne rezystory w RGB)

## Czego brakuje

- [ ] Tryb +3 (CP/M, 3" floppy)
- [ ] Obsługa AY-3-8912 (dźwięk)

## Ekran startowy (REAL)

```
Sinclair BASIC 128
(C) 1986 Sinclair Research Ltd

128K BASIC

0
```

Biały tekst na czarnym tle. Ekran memory testu zlicza do 131072 podczas bootowania.
